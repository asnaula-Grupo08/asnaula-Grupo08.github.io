Listas de nodos JavaScript HTML DOM

Veámos qué son exactamente las listas de nodos y cómo podemos manejarlas de una forma cómoda y sin equívocos.

  • NodeLists

    Un aspecto importante a tener en cuenta es que, como cuaquier otro método que interactúa con el DOM, las listas de nodos no pertenecen a la especificación Javascript, sino a la API que los navegadores proporcionan para que Javascript interactúe con el modelo de objetos en el documento.

  • NodeLists y Arrays

    Una lista de nodos es una colección ordenada de elementos. Esto recuerda de forma directa al concepto de arrays; de hecho las similitudes son muchas.

El objeto HTML DOM NodeList

Un NodeListobjeto es una lista (colección) de nodos extraídos de un documento. Un NodeListobjeto es casi lo mismo que un HTMLCollectionobjeto.Todos los navegadores devuelven un objeto NodeList para la propiedad "childNodes". La mayoría de los navegadores devuelven un objeto NodeList para el método "querySelectorAll()".
